# 온보딩 (비개발자용)

git을 몰라도 된다. 터미널을 열 필요도 없다. 아래 3단계면 끝.

## 1. 플러그인 설치

- (내부 배포) 관리자가 준 `obsidian-git-sync` 폴더를 vault의 `.obsidian/plugins/`에 넣거나 BRAT로 설치.
- Obsidian 설정 → 커뮤니티 플러그인에서 **obsidian-git-sync** 활성화.

## 2. "연결 설정" 위저드

플러그인 설정 화면의 **[연결 설정]** 버튼을 누르면:

1. 이 기기 전용 SSH 키가 자동으로 만들어진다. (당신의 개인 `~/.ssh`는 건드리지 않는다.)
2. 화면에 **공개키 한 줄**과 **[복사]** 버튼이 나온다.
3. 서버 주소(remote URL)를 입력한다. (관리자에게 받는다. 예: `ubuntu@152.69.239.170:vault.git`)

## 3. 공개키 전달

- 복사한 공개키를 **관리자에게 전달**한다 (메신저/메일 아무거나).
- 관리자가 등록하면 동기화가 시작된다.

---

이후로는 **전부 자동**이다.

- 노트를 쓰면 알아서 저장·동기화된다. 상태바에 "동기화됨"이 뜬다.
- 다른 사람(및 hermes)이 작업 중인 내용은 사이드 패널에서 확인할 수 있다.
- 확정하고 싶을 때 **[저장]** 버튼을 누르면 공식본(main)에 반영된다.

## 문제가 생기면

- 상태바에 "오류"가 뜨면 잠시 후 자동 재시도한다. 계속 뜨면 공개키 등록 여부를 관리자에게 확인.
- 편집은 오류 중에도 계속 가능하다 — 동기화만 잠시 미뤄질 뿐 데이터는 로컬에 안전하다.
